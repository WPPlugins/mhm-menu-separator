== Description ==

Add customisation to allow line and text-only entries to the output of WordPress' wp_nav_menu function.

Edit your menu in the “Appearance” section of wp-admin. Add a custom link entry, then use --- (three dashes)
as link text for a horizontal line or use # as a URL for an unlinked menu entry.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
